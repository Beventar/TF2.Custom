# Woolhud
This is a recreation of the HUD Woolen is seen using in his YouTube videos and in his Twitch streams if you're into watching those.

# Important Information
This HUD is for 16:9 resolution users so if you play with 4:3 IN GAME, nevermind that you have 16:9 monitor, you have to play IN GAME 16:9 


# More Info
This HUD has some customizations like from older version of HUD from older Woolen videos, and you can use version of HUD from newer Woolen videos
I made somethings more cleaner than what Woolen actually has.

Note: This is buggy and glitchy version. Woolen's HUD is glitchy and buggy.
